Git commands and descriptions

1. git Info

>

To initialize git in our prouect folder

2. git Status

To check or show the status of tracked and untracked files from git cache

3. git add filename

To add single file to git cache which are untracked

4. git add

To add all the untracked files to git cache

5. git rm --cached filename

To remove the last added untracked files from the git cache

6. git commit -m "message"

generates the committed id with comit message to make ready to push

7. git remote add origin repo link

adds the remote server to our local server

8. git push -u origin branchname

to push the committed code/file to git remote server with branch name

9. git show

to view the file in console
